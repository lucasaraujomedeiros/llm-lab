# LLM Lab — Introdução Prática a Modelos de Linguagem

## Descrição

Este projeto tem como foco a exploração prática e sistemática de **Large Language Models (LLMs)** a partir de uma abordagem “bottom-up”, onde os principais componentes de um modelo moderno são construídos explicitamente, sem abstrações de alto nível.

A proposta parte da implementação de um modelo inspirado na arquitetura GPT, com o objetivo de expor os mecanismos fundamentais que sustentam modelos de linguagem atuais — incluindo tokenização, representações vetoriais, mecanismos de atenção e processos de treinamento baseados em predição de próxima palavra.

Diferente de abordagens puramente teóricas ou baseadas em uso de bibliotecas prontas, o projeto enfatiza a **compreensão estrutural e operacional** dos modelos.

Essa abordagem permite tratar o desenvolvimento de LLMs como um processo **experimental e mensurável**, e não apenas como uma aplicação de técnicas prontas.
